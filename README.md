# Maze Solver

A Python project that generates and solves mazes visually using graph traversal algorithms and recursive backtracking.

Built as part of the [Boot.dev](https://www.boot.dev?utm_source=chatgpt.com) backend and computer science curriculum.

---

## Features

- Procedurally generates random mazes
- Visually renders the maze
- Solves mazes automatically
- Recursive backtracking generation
- Pathfinding visualization
- Object-oriented Python design
- Simple graphics-based UI

---

## Technologies Used

- Python 3
- Tkinter (or custom graphics library, depending on implementation)
- Object-Oriented Programming
- Recursion
- Graph traversal algorithms

---

## Concepts Practiced

This project focuses on several important programming and computer science concepts:

- Recursion
- Depth-First Search (DFS)
- Graph traversal
- Backtracking algorithms
- Data structures
- Coordinate systems
- GUI rendering
- State management
